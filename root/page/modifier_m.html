JavaScript Regular Expression

The m Modifier

Do a multiline search for "is" at the beginning of each line in a string:

    
        
            
            <p id='demo'>
</p>
            let text = `Is this
            all there
            is`
            
            let pattern = /^is/m;

            let result = text.match(pattern);
            
            document.getElementById("demo").innerHTML = result;
